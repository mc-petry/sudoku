# Sudoku

HTML5 sudoku implementation

[Demo](http://mc-petry.name/projects/sudoku/)

Technologies:

* [React](https://github.com/facebook/react) via JSX
* [Redux](https://github.com/rackt/redux)


## Development

### Build project

```sh
$ gulp
```

### Watchify

```sh
$ gulp watch
```

### Tests

```sh
$ npm test
```
